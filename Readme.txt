How install app?
-You only need Maven.
-Maven Update.
-Maven install.
-Run PatentParserApplication, it will run as a Server.

Used Technologies:
-H2 database, not need instalation, can generate database only running program and is so good database to create POCs.
-I use JPA because is so clean for read code, and so easy to implementation.
-Thymeleaf because is so easy to do quickly front, and in this challenge the importante part is background.
